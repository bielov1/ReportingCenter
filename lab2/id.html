Селектори ідентифікаторів в CSS

Селектори ідентифікаторів використовуються для вибору елементів HTML за їх унікальними ID.

використання ідентифікатора:

  • В HTML-коді додати елементу атрибут 'id' з унікальним значенням.
  • В CSS-коді перед значення 'id' додати символ '#'

    • Приклад:

    html:

    <form id="my__form" method="post"></form>

    CSS:

                                #my__form {
                                border: white 1px solid;
                                border-radius: 23px;
                                width
                            }
