Таблиці:

Елементи, які можуть або повинні використовуватись в таблицях:

Назва елементу Значення елементу
<caption> Тег <caption> в HTML використовується для надання заголовка або заголовка таблиці. Він дозволяє пояснити вміст таблиці, щоб користувачі краще розуміли, що містить ця таблиця.
<colgroup> Тег <colgroup> в HTML використовується для групування стовпців у таблиці. Він дозволяє застосовувати стилі або атрибути до групи стовпців, а не до кожного стовпця окремо.
<thead> Тег <thead> в HTML використовується для визначення заголовка таблиці. Він дозволяє виділити рядоки таблиці, які містять заголовки стовпців.
<tbody> Тег <tbody> в HTML використовується для визначення основного вмісту таблиці, тобто рядків з даними. Він відокремлює вміст таблиці від її заголовка та підсумку.
<tr> Тег <table> row в HTML визначає рядок в таблиці. Він містить один або кілька елементів <th> (заголовок стовпця) або <td> (дані комірки).
