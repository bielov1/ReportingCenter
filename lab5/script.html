Вставка(тег script)

Використання елементу <script> в HTML надає можливість інтегрувати код JavaScript прямо у веб-сторінку. Завдяки цьому розробники можуть створювати та запускати скрипти, які здатні динамічно змінювати контент сторінки, забезпечувати інтерактивність з користувачем та виконувати інші динамічні дії.

Приклад:

<body>

.... <script> alert("Я скрипт"); </script> </body>

Важливо використовувати тег <script> в кінці тегу <body>, щоб сторінка спочатку завантажила теги HTML та CSS, і лише в кінці js.
