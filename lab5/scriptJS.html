Застосування cценарію JavaScript у власному сайті

Скролінг іграшок в "в тренді"

Застосування сценарію JavaScript для скролінга іграшок надає сайту більшу динамічність і інтерактивність. Це дозволяє користувачам зрозуміти, які іграшки зараз в тренді та що можна вибрати швидко дитині в подарунок. Плавний скролінг створює приємний візуальний єфект і покращує загальне враження сайту.


Вирізка з HTML коду

<div class="hot_sellers_items"> <button class="scroll-left" onclick="left()">&lt;</button> ... <button class="scroll-right" onclick="right()">&gt;</button> </div>

Вирізка з CSS коду

                        .scroll-left,
                        .scroll-right {
                          overflow: hidden;
                          position: absolute;
                          background: rgba(0, 0, 0, 0.5); 
                          border: none;
                          color: #fff;
                          font-size: 24px;
                          cursor: pointer;
                          padding: 10px;
                          z-index: 1;
                        }
                        
                        .scroll-left {
                          left: 0;
                        }
                        
                        .scroll-right {
                          right: 0;
                        }

Вирізка з JS коду

                        function left() {
                            const hotSellersItems = document.querySelector('.hot_sellers_items');
                            const scrollAmount = 200;
                            hotSellersItems.scrollBy({
                                left: -scrollAmount,
                                behavior: 'smooth'
                            });
                        }
                        
                        function right() {
                            const hotSellersItems = document.querySelector('.hot_sellers_items');
                            const scrollAmount = 200;
                            hotSellersItems.scrollBy({
                                left: scrollAmount,
                                behavior: 'smooth'
                            });
                        }
